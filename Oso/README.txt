Autor: Juan Pablo Jimenez Arenas
Probado en: NetBeans 22 de la VDI com java 17
Juego oso multijugador/SOLITARIOvsIA con chat
Instrucciones  
Todos los .jar se ejecutan con "java -jar ejecutable.jar", sin necesidad de meter niguna variable.
--------------  
	Para Jugar en consola solo:
		 Ejecutar OsoSolitarioConsola del paquete oso.ui

	Para Jugar en GUI solo:
		Ejecutar OsoSolitarioGUI del paquete oso.ui

	Para usar el chat:
		Ejecutar servidor TcpChatSeverMultiThread del paquete oso.net. Ejecutar muchos clientes ChatClientGUI del paquete oso.ui.

   Para jugar al juego completo:
      Ejecutar servidores TcpOsoServerMultiGames y TcpChatSeverMultiThread del paquete oso.net. Ejecutar un cliente OsoGUI y un cliente ClienteOsoBot del paquete oso.ui.
      Ejecutar servidores en la carpeta servidores Jimenez_Arenas_JuanPablo_OSO_TcpChatServerMultiThread.jar y Jimenez_Arenas_JuanPablo_OSO_TcpOsoServerMultiThread.jar. 	  Ejecutar un cliente Jimenez_Arenas_JuanPablo_OSO_OsoGUI.jar
      y un cliente Jimenez_Arenas_JuanPablo_OSOClienteOsoBot.jar de la carpeta Clintes. Tambien puedes jugar en local contra el BOT sin necesidad de abrir el servidor 		  dandole a Un Jugador en  OsoGUI del paquete oso.ui o 
      Jimenez_Arenas_JuanPablo_OSO_OsoGUI.jar.
